================================================================================
KEY FUNCTIONS NEEDED
================================================================================

1. showStep(stepNumber) 
   - Hide all steps, show requested step
   - Update progress indicators

2. validateCurrentStep()
   - Check if required fields filled
   - Enable/disable continue button

3. setupButtonGroup(containerId, fieldName)
   - Handle option button clicks
   - Update visual state
   - Update form data

4. submitForm()
   - POST to /api/recommendations
   - Show loading state
   - Display results

5. togglePlant(plantId)
   - Add/remove from selectedPlants Set
   - Update visual state
   - Update plant count

6. renderPlants()
   - Create plant card HTML for each plant
   - Attach event listeners

7. generateImage()
   - POST to /api/generate-image
   - Show loading state
   - Display generated image

8. updatePlantCount()
   - Update "X of Y selected" display

9. updateImageButton()
   - Update button text with count
   - Enable/disable based on selection


================================================================================
API INTEGRATION POINTS
================================================================================

1. Form Submission:
   POST /api/recommendations
   Body: { address, direction, water, maintenance, gardenType }
   Response: { zone, season, plants[], timeline[] }

2. Image Generation:
   POST /api/generate-image
   Body: { plants[], direction }
   Response: { imageUrl, plantCount }

3. Photo Upload:
   POST /api/upload-photo
   Body: FormData with file
   Response: { photoUrl }
